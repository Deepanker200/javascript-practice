# Copying in JS
    - If we assign an object to another variable, both variables point to the same memory location. (Vice versa)
    - A shallow copy creates a new object, but nested objects are still shared. (Vice versa for nested)
    - A deep copy creates a completely independent object, including nested objects. (No Vice Versa)
